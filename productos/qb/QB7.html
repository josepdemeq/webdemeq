Serie QB7

Medidores de Espesores por Efecto Hall

La mejor opción

Capaz de medir todo tipo de recipientes de geometrías complejas y paredes de espesores muy finos difíciles de medir con otros métodos. Idóneo para cuantificar el espesor de botellas PET, tambores plásticos, latas de aluminio, ampollas de vidrio y básicamente cualquier parte o pieza hecha de un material no ferroso incluyendo gomas, materiales compuestos y laminados.

Preciso, versátil y fácil de usar

Es una herramienta desarrollada para verificar espesores en producción y para control de calidad en todo tipo de materiales no magnéticos con espesores de hasta 22mm.

Almacenamiento de las mediciones y conexión al PC

Almacene 32000 datos que se pueden transferir a una PC vía USB usando nuestro software dmq DataCenter (sin costo adicional) o con un cable RS232.

Listo para usar

Nuestros equipos se entregan listos para usar. Presentados en un maletín de alto impacto para guardado y transporte, en el interior encontrará:

  • Unidad electrónica QB7
  • Cable de conexión para el sensor
  • Pendrive con software dmq DataCenter y manual
  • Sensor por efecto hall con capuchón de titanio
  • Cable USB
  • Base para el sensor de 2 partes
